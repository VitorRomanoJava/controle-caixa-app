<h1>Controle de Caixa</h1>

<p>
Aplicação para controle de fluxo de caixa desenvolvida para registrar entradas, saídas e acompanhar o saldo mensal de forma simples e organizada.
</p>

<p>
O projeto faz parte da minha evolução no desenvolvimento de aplicações modernas, explorando React, TypeScript e integração com banco de dados.
</p>

<h2>Tecnologias</h2>

<h3>Frontend</h3>
<ul>
<li>HTML5</li>
<li>CSS3</li>
<li>TypeScript</li>
<li>React</li>
<li>Vite</li>
<li>TailwindCSS</li>
</ul>

<h3>Backend / Dados</h3>
<ul>
<li>PostgreSQL</li>
<li>Supabase</li>
</ul>

<h3>Desktop</h3>
<ul>
<li>Tauri</li>
</ul>

<h2>Funcionalidades</h2>
<ul>
<li>Registro de entradas</li>
<li>Registro de saídas</li>
<li>Visualização do saldo mensal</li>
<li>Metas financeiras</li>
<li>Notificações</li>
<li>Cofrinhos para objetivos financeiros</li>
</ul>

<li><strong>Área principal</strong> com as páginas do sistema</li>
</ul>

<p>Principais módulos:</p>

<ul>
<li>Fluxo</li>

</ul>

<h2>Executando o projeto</h2>

<p>Clone o repositório:</p>

<pre><code>git clone https://github.com/seuusuario/controle-caixa.git</code></pre>

<p>Entre na pasta:</p>

<pre><code>cd controle-caixa</code></pre>

<p>Instale as dependências:</p>

<pre><code>npm install</code></pre>

<p>Execute o projeto:</p>

<pre><code>npm run dev</code></pre>
